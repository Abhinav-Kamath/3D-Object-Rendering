key bindings
o i -> init()
o t -> toggle between 3D view and topView
o a -> animate
o x -> rotate camera around X-axis
o y -> rotate camera around Y-axis
o z -> rotate camera around Z-axis
o X -> rotate object around X-axis
	Use arrowleft and right to do the rotation
o Y -> rotate object around Y-axis
	 Use arrowleft and right to do the rotation
o Z -> rotate object around Z-axis
	Use arrowleft and right to do the rotation
o + -> scale up
o - -> scale down
o ArrowUp -> speed up animation
o ArrowDown-> slow down animatio
